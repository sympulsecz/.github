# .github

Organization-wide GitHub defaults for Sympulse: the pull request template, contributing guide,
and support pointers that apply to every repo that doesn't define its own.

This repository is intentionally **public**. GitHub only serves default community health files
from a public `.github` repo, so this one has to be public for the defaults to take effect
org-wide. It contains no code or sensitive data. The member-only org profile lives in
`.github-private`.

Why these conventions exist is documented in `sw-standards` (team members):
https://docs.dsai.sympulse.cz
