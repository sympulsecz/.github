# .github

Organization-wide GitHub defaults for Sympulse: the pull request template, contributing guide,
and support pointers that apply to every repo that doesn't define its own.

This repo is public so GitHub can serve these files as [org-wide defaults](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

Why these conventions exist is documented in `sw-standards` (team members):
https://docs.dsai.sympulse.cz
