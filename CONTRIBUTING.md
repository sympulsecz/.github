# Contributing

How we work is documented in our standards (team members):
https://docs.dsai.sympulse.cz

The short version:

- Branch off `main` using `feature/`, `fix/`, or `chore/`.
- Write [conventional commits](https://docs.dsai.sympulse.cz/version-control.html); keep each
  commit atomic and link the ticket (`Fixes GH-123` / `Closes JIRA-456`).
- Keep PRs small, review your own diff first, and push early as a draft.
- Experimental (`x-`) repos have relaxed requirements.

Questions? Ask in `#developers` on Slack.
